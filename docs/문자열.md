# 문자열
`Neko++`에서 문자열을 표현하기 위해서는 `"` 또는 `'`으로 문자열을 열고 닫습니다.
또한, `🍞`을 사용해 `식빵끼워넣기` 기능을 사용할 수 있습니다.


## 문자열 리터럴

|리터럴|설명|
|:---|:---|
|`\꾸욱`|줄바꿈을 합니다.|
|`\꽈아악`|탭만큼 띄웁니다.|

<br>

---

## 예제 코드

* 일반 문자열
```sh
"냥~"
'냥~'
```

* 줄바꿈하기
```sh
"냥~\꾸욱"
```

* 탭하기
```sh
"냥~\꽈아악"
```

* 식빵끼워넣기
```sh
먁꺅 "냥~"
"{}".🍞(먁)
```
