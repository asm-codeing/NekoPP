# 입력문
`Neko++`에서 귀여운 사람에게 문자를 입력받기 위해서는 `냐옹`, `냐아옹`, `냐아아옹` 키워드를 사용합니다.

## 키워드 사양

|키워드|설명|반환형|
|:---|:---|:---:|
|`냐옹`|문자열로 입력을 받습니다.|`문자열`|
|`냐아옹`|정수로 입력을 받습니다.|`정수`|
|`냐아아옹`|실수로 입력을 받습니다.|`실수`|

## 키워드 세부 사양

|매개|설명|자료형|기본 값|
|:---|:---|:---|:---:|
|`-`|어떠한 매개도 받지 않습니다.|`-`|`-`|

<br>

---

## 예제 코드

* 문자열 입력받기
```sh
먁꺅 냐옹
```

* 정수 입력받기
```sh
먁꺅 냐아옹
```

* 실수 입력받기
```sh
먁꺅 냐아아옹
```
